# BOTH Components Expansion — Draft

**Date:** 2026-06-17
**Status:** Draft (in discussion)

## 维持 TRIGGER_ONLY

**完成圈数**
- 当 T：每完成 N 圈后激活

## 维持 BOTH（不变）

**治愈**
- 当 T：每触发 N 次治愈效果后激活
- 当 E：立即恢复一定量生命值

**反射**
- 当 T：每触发 N 次反射效果后激活
- 当 E：下次受到伤害时，将该伤害的一定比例反弹给攻击者

## Trigger-only → BOTH

**受击**
- 当 T：每受到 N 次伤害后激活
- 当 E：对自身造成一定量真实伤害，不致死

**击杀**
- 当 T：每击杀 N 个敌人后激活
- 当 E：斩首——对当前敌人造成等同于其剩余生命值一定比例的额外伤害

**经过**
- 当 T：每经过 N 个地块后激活
- 当 E：立即对当前地块额外触发一次规则

**低血**
- 当 T：生命值低于 30% 时每秒激活
- 当 E：主动扣除自身一定量生命值，不致死

**满血**
- 当 T：生命值满时每秒激活
- 当 E：当前身上所有类型的叠层各额外增加 1 层（没有的不增加）

**规则触发**
- 当 T：每有 N 条其他规则触发后激活
- 当 E：所有规则槽的触发计数各 +1，不直接触发，只加速蓄力

## Effect-only → BOTH

**护盾**
- 当 T：每 N 次护盾吸收伤害后激活
- 当 E：获得一定量护盾值，先于生命值承受伤害

**减伤**
- 当 T：每 N 次敌人伤害被减免后激活
- 当 E：永久降低敌人对你造成的伤害，可叠加，上限 80%

**吸血**
- 当 T：每 N 次吸血回血后激活
- 当 E：永久提升每次攻击后的回血比例，可叠加

**强化**
- 当 T：每 N 次强化被消耗后激活
- 当 E：获得 1 层强化，下一条规则触发时效果翻倍并消耗

**增伤**
- 当 T：每 N 次增伤效果打出伤害后激活
- 当 E：获得若干层增伤，每层提升下次攻击伤害，每圈衰减

**蓄能**
- 当 T：每 N 次蓄能释放后激活
- 当 E：积累若干层蓄能，达到上限时自动释放造成爆发伤害

**灼烧**
- 当 T：每 N 次灼烧伤害跳动后激活
- 当 E：对当前敌人施加灼烧，每秒持续造成伤害

**侵蚀**
- 当 T：每 N 次侵蚀伤害跳动后激活
- 当 E：对当前敌人施加侵蚀，每秒持续降低其防御并造成伤害
