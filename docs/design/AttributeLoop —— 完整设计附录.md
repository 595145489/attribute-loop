# AttributeLoop —— 完整设计附录

> 本篇为《AttributeLoop —— 游戏设计文档》的展开，补充全部组件表、阶段数值、战斗与经济细节，末尾附可选的拍卖支线。

**AttributeLoop** 是一款单地图、渐进式加难的 Roguelike，灵感来自《循环英雄》。玩家在环形轨道上自动行走，与敌人战斗，并**把规则从敌人身上撕下来**，组装成改写世界的"活规则"。

| 项目 | 内容 |
|---|---|
| 类型 | 单地图渐进式 Roguelike / 自创 build 自动战斗 |
| 平台 | HTML5（网页），Godot 4 引擎 |
| 单局时长 | 约 30–35 分钟 |
| 视角 | 俯视角 |

> 一句话定位：一款以"自创规则"为核心的 Roguelike——从敌人身上撕下两段式规则碎片，任意触发器配任意效果，种进世界让其永远生长，最后活过你自己一手构筑出的裁决圈。

---

## 1. 设计支柱

1. **自创自由（心脏）**——力量是自己写出来的，不是被分配的。没有职业、技能树、预制 build；自由是默认状态，其余设计都是让这份自由成为有意义的选择。
2. **组件三用途之争**——每个好组件都想同时放进玩家槽、地块、祭坛三个地方中的两个，而你做不到。策略深度正来自这个事实。
3. **规则约占玩家力量 90%**——裸属性只提供存活基线。第 3 阶段起，裸装玩家在战斗窗口内根本杀不死敌人，规则从可选变为强制。
4. **软反蹲与可读数值**——血量 200，15 点治疗一眼是 7.5%；激怒线性增长（每层 +30%），不是指数，赖皮流最终会输但输得可预测。

## 2. 核心玩法 —— 规则构筑沙盒

### 2.1 语法：触发器 + 效果

每条规则恰好两个槽：**触发器**（何时开火）加**效果**（做什么）。槽位强制——放触发器槽的组件以 T 行为结算，放效果槽的以 E 行为结算；同一个两用组件放哪个槽就扮演哪个角色。

### 2.2 词表：17 个组件

17 个组件中 16 个既能当触发器（T）、也能当效果（E），仅"完成圈数"只有 T。值 N 表示该事件发生 N 次后规则开火，随后计数器重置。

| 组件 | 作为触发器 T（何时开火） | 作为效果 E（做什么） |
|---|---|---|
| 受击 | 每承受 N 次伤害 | 对自身造成真实伤害（不会致死，最低 1 血），并计入一次受击事件 |
| 击杀 | 每击杀 N 名敌人 | 对当前敌人造成等于其当前生命一定百分比的伤害 |
| 经过 | 每经过 N 个地块 | 立即额外触发一次当前地块的规则 |
| 规则触发 | 每有 N 条其他规则开火 | 所有玩家规则槽的触发计数各 +1（仅加速蓄能，不立即开火） |
| 低血 | 血量低于 30% 时每 N 秒 | 对自身造成较轻真实伤害（不会致死，最低 1 血） |
| 满血 | 处于满血时每 N 秒 | 所有已大于 0 的层数（蓄能 / 增伤 / 强化）各 +1 |
| 治愈 | 每发生 N 次治疗 | 恢复固定生命 |
| 护盾 | 每护盾吸收 N 次伤害 | 获得吸收伤害的护盾 |
| 反射 | 每发生 N 次反射 | 下一次受击按设定比例反弹给攻击者 |
| 减伤 | 每次减伤结算 N 次 | 每层 −10% 受到的伤害，可叠加，上限 80% |
| 吸血 | 每吸血治疗 N 次 | 永久提升每次攻击的吸血比例，可叠加 |
| 强化 | 每消耗 N 次强化 | 下一个非强化效果 ×(1 + 层数 × 0.5) |
| 增伤 | 每次增伤命中 N 次 | 每层 +10% 玩家伤害，每圈衰减 |
| 蓄能 | 每释放 N 次蓄能 | 累积蓄能层数，满后自动释放为爆发攻击 |
| 灼烧 | 每次灼烧跳伤 N 次 | 施加灼烧层，每秒造成持续伤害 |
| 侵蚀 | 每次侵蚀结算 N 次 | 永久削减当前敌人最大生命 |
| 完成圈数 | 每完成 N 圈 | 暂无 |

组件数值按掉落阶级（普通 / 稀有 / 史诗）分档，越后期、越高阶的敌人掉落越强、触发越频繁（N 越小）。

### 2.3 规则的三个放置点

- **玩家槽（战术层）**：起始 2 槽，上限 5。任意时机免费替换，被换组件回背包。效果以基础值结算、不成长，服务于当下。
- **地块（投资层，自由度在此复利）**：13 个地块（1 祭坛 + 12 行走），共 23 个规则槽，分布不均。永久，移除需付递增金币且销毁。效果随经过次数成长：`地块效果 = 基础值 × (1 + 成长率 × 经过次数)`，设上限封顶，再加祭坛加成、强化倍率。约 30 圈后治愈地块约为基础值的 5.5 倍。地块颜色随经过次数由灰→蓝→黄，让玩家看见投资成熟。非战斗效果每次经过都触发；战斗效果仅当地块有敌时触发。
- **祭坛（献祭层）**：第 13 个特殊地块。每阶段要求投入固定数量组件，填满激活后组件永久消耗，转化为永久属性加成并推进阶段。祭坛加成整局叠加——早期献祭永久分红。

### 2.4 为什么这个沙盒是高自由度的

1. **两用组件**——16 个可坐任一槽位，敌人身上威胁你的"受击"（自伤），到你手里可变成防御型触发器。语法没有死字。
2. **自指链条是特性**——"治愈触发 → 治愈效果"开火会发出"规则开火"事件，被"规则触发"捕获，治疗本身又回头触发"治愈触发"。循环被允许，靠槽位成本、激怒时钟、最小触发阈值 2 平衡。
3. **成长因放置而异**——同一个治愈，穿身上固定 15；种地里今天 15、三十圈后 82。价值随放哪、何时放而变。
4. **玩家槽即时可换**——任意时机免费替换。遇弱敌时压住节奏，换上蓄能词条攒满层数与触发次数，把这波小怪当下一场硬仗的装弹期。

### 2.5 涌现的 build 流派

游戏从不告诉玩家这些流派存在，它们从语法中自然长出：

- **被动治疗引擎**——"经过 + 治愈"种满行走地块，每圈自动回血。
- **反射坦克**——"受击 + 反射"装玩家身上，挨打反伤。
- **处决核弹**——"受击 + 击杀（按当前生命百分比）"，受击即按敌人当前血%造伤，专杀坦克。
- **灼烧侵蚀消耗**——战斗地块上堆灼烧（持续伤害）+ 侵蚀（削最大生命），绕开裸血裸伤。
- **自循环续航**——串联互相再触发的治疗与护盾，前期强，但线性激怒最终压垮它——软天花板，刻意为之。

那片涌现空间，才是产品本身。

## 3. 核心循环

每圈是同一个循环的变奏：**自动行走 → 遭遇敌人 → 自动战斗（规则开火）→ 从尸体剥取组件 → 三选一分配（玩家槽 / 地块 / 祭坛；背包满则花金币删除多余组件）→ 继续行走 → 圈末结算（临时属性衰减、世界压力推进）→ 填满祭坛则阶段推进并获 buff，未填满则阶段照推进但无 buff。**

战斗刻意自动，好让真正有趣的决策——构筑规则集——发生在战斗之间。

## 4. 成长弧线 —— 阶段与裁决圈

阶段让自由度有截止线：没有不断升级的压力，最优玩法就是永远种地。

| 阶段 | 名称 | 圈数窗口 | 每圈敌人数 | 祭坛需求 | 引入 |
|---|---|---|---|---|---|
| 1 | 觉醒 | 8 | 2–3 | 2 | 汲取者、守卫者 |
| 2 | 涌动 | 7 | 2–3 | 3 | 急袭者 |
| 3 | 侵蚀 | 6 | 3 | 4 | 复制者 —— 规则转为强制 |
| 4 | 失衡 | 5 | 3–4 | 5 | 先驱者 |
| 5 | 裁决前夜 | 4 | 3–4 | 6 | — |
| 6 | 裁决圈 | 存活 5 圈 | 5–6 | — | 满配敌人 ×3.05 |

敌人属性 = 基础值 × (1 + (当前阶段 − 1) × 0.25)，第 6 阶段为 ×3.05。第 3 阶段起裸攻击无法在 8–12 秒内击杀，规则转为强制；第 5 阶段无 build 则必死。

- **世界压力（兜底）**：未在窗口内填满祭坛，阶段照样推进但无 buff——软失败，落后于 buff 曲线会雪球式滚向准备不足地进入裁决圈。
- **Boss 圈**：阶段 1–5 推进前最后一格生成 Boss（生命 ×2、攻击 ×2、规则对 +2），祭坛推进即触发，无法跳过——直接检验 build 能否扛住下一阶段数值。
- **裁决圈（期末考）**：阶段 5 祭坛填满、Boss 击败后进入。每圈 5–6 名敌人分布 12 地块，全按 ×3.05、五型全开、每只 4 对规则（满配），无压力计时。**胜**：连续存活 5 圈；**败**：血量归零。裸装数学上必死（一只阶段 6 汲取者裸装要约 24 秒才能杀死，远过激怒悬崖）。

| 结局 | 条件 |
|---|---|
| 失败 | 玩家血量任意时刻归零 |
| 真正胜利 | 在裁决圈存活 5 圈 |
| 压力螺旋 | 反复触发世界压力而无祭坛 buff，功能性导向失败 |

## 5. 战斗 —— 你 build 的检验场

战斗刻意低交互（自动结算），好让有趣的决策发生在战斗之间。

**节奏锚点**：每场普通战斗裸装 8–12 秒。这是神圣锚点——更短则规则无意义，更长则变放置模拟。

**伤害结算**：

- 玩家伤害 = (基础 10 + 加成) × (1 + 增伤层数 × 0.1) × 蓄能层数 × 减伤免税 − 敌方护盾（护盾优先吸收）
- 敌人伤害 = 敌人攻击 × (1 + 激怒层数 × 0.30) × 减伤免税 − 玩家护盾
- 命中时：吸血按比例回血、反射反弹、灼烧每秒跳一次持续伤害

**保持诚实的子系统**：

- **激怒（反蹲）**：开战 10 秒后敌人 +30% 伤害，之后每 2 秒再 +30%，线性。不杀敌的续航循环最终会死——这正是支柱四和自指链条不破坏游戏的保证。
- **灼烧**：灼烧层数 × 2 每秒伤害。
- **减伤**：每层 −10%，上限随阶段温和成长（最多 8 有效层），永远无法归零伤害。
- **圈末衰减**：每圈结束护盾降至 65%、减伤与增伤层剥落，临时力量不跨圈滚雪球；永久力量（地块规则、祭坛加成）不受影响。

**敌人人格**——每种敌人是带人格的规则调色板，打法不同，以身教教会玩家构筑空间：

| 敌人 | 人格 | 掉落倾向 |
|---|---|---|
| 汲取者 | 续航——治疗与吸取 | 治疗、吸血 |
| 守卫者 | 坦克——难以击破 | 护盾、减伤、反射 |
| 急袭者 | 爆发——快速而脆弱 | 增伤、蓄能、灼烧 |
| 复制者 | 连击——规则链 | 规则触发、强化、侵蚀 |
| 先驱者 | 全能——权重均匀 | 任何东西（期末考） |

先驱者权重完全均匀，掉什么全凭运气，是构筑空间的压力测试。

## 6. 经济与背包管理

金币仅来自击杀敌人，按阶段分档成长。**背包上限（起始 12 格）是分配张力的第一道闸**——剥取免费但背包有上限，满了就必须处置：装槽、种地、献祭，或删除。**删除**是清理背包的唯一回收手段，花金币移除组件，成本随本局删除次数递增。它不是放置点，只是不被垃圾组件卡死的排气阀；越删越贵，仍推向"一次就分好"。

## 7. 世界观与叙事

两层刻意融合：

- **表层（玩家所见）**：两个陌生人每晚共享同一个梦——一条柔和发光的环形小径，中心一座灯常亮的小亭。他们从不同时抵达，只能看见对方留下的东西。敌人不是怪物，而是阻隔两个尚未相遇之人的那些东西。
- **机制层（规则构筑的隐喻）**：折纸。组件是折纹，规则是折法，世界是正在被折出形状的纸——整款游戏都在讲把碎片折叠成活的结构。

梦境是玩家感受到的故事，折纸是机制传授的隐喻，二者互不矛盾。

### 逐阶段故事

全程温暖，即便酸楚。

- **第 1 阶段 · 觉醒**：初临。一条发光的小径。一座灯亮的亭。一朵看不见的人放的花。你走进去，你不知道为什么。
- **第 2 阶段 · 涌动**：路边的石子被排成了一个太刻意、不可能是偶然的形状。你把自己的排成一个问号，下次再来，旁边多了一个感叹号——你留下的问号，已经变成你们之间的一根线。
- **第 3 阶段 · 侵蚀**：亭壁上开始出现字——你的和对方的。怀疑侵蚀着确定：这一切是真的吗？你是不是只在自言自语？
- **第 4 阶段 · 失衡**：你开始期待入睡。醒着的时间变得模糊，梦里的小径反而更清晰。你正爱上一个从未见过的人。
- **第 5 阶段 · 裁决前夜**：怀疑被打碎，你们都开始诚实书写。亭壁再无空隙——你们之间的东西是真的，那是这个梦运行的唯一法则。
- **第 6 阶段 · Boss（裁决前夜）**：亭灯第一次熄灭。墙上潦草而不均的字迹，不像他平日细致的刻痕——"我可能回不来了。"你站了很久，然后在旁边刻下："我会去现实世界找你。"

### 裁决圈 —— 寻找

梦结束了。你醒来，除了碎片一无所有：歪斜的字迹、某人窗前夜里可见的一座塔、一种你到哪都认得出的摆石方式。裁决圈是寻找——没有地图，只有你记得的，和你愿意走多远。找到是一种答案，知道你找过是另一种，两种都不错。这个故事不是悲剧——对方是真实的，在某处也在找；梦没有起源故事，它存在，这就够了。

## 8. 设计边界与未来可能

明确不在当前游戏中，列出以防范围蔓延：规则语法的第三个动词槽；变异系统（错型组件强塞槽位）；跨局组件迁移或 meta 进度；多地图或关卡结构；五种以外敌人类型；裁决圈存活圈数调参（当前 5 圈）。

## 9. 扩展系统 —— 拍卖（残市）

> 可选支线，非核心循环。玩家可以完全不碰拍卖通关；它给喜欢第二条成长轴的玩家一个额外金币去向。

拍卖是与规则系统争抢金币的第二条成长轴。规则是主轴，拍卖只是让多余金币有处可去——玩家也可干脆不竞拍，把金币全留给删除。

| 服务 | 效果 |
|---|---|
| 战意磨砺 | +2 基础攻击（永久） |
| 筋骨强化 | +15 最大生命，并回复等量 |
| 迅捷折纸 | −0.05 秒攻击间隔（下限 0.2 秒，最多 12 次） |
| 强化潜能 | +1 强化层数上限 |
| 装备槽扩容 | +1 玩家规则槽（上限 5） |
| 道具栏扩容 | +1 道具栏容量 |
| 词条改写 | 提升某组件效果值（或调整触发器 N） |
| 词条融合 | 合并两个同型组件：结果 = (a + b) × 0.8 |
| 敌人赦免 | 接下来 3 只指定敌人自动掉落（免战） |
| 删除特赦 | 下一次删除免费且不计入次数 |
| 压力延缓 | 世界压力窗口 −1 圈 |

两个 AI 买家每场拍卖与玩家竞争，刻意可读（显示兴趣等级：无/低/中/高），让竞拍成为心理博弈而非黑箱：**影子甲（激进型）**每圈花约 75% 预算、偏好裸属性升级；**影子乙（蓄势型）**攒够 200 金且偏好服务出现时梭哈（约 85%）。玩家收入大致与幻影持平，不可能什么都抢得到。

---

## 结语

> AttributeLoop 是一款关于"写规则"的游戏。组件是字母，触发器加效果是语法，三种放置是寄存器，战斗是告诉你程序跑不跑得起来的编译器。其他每个系统——经济、阶段、裁决——都为确保写出好规则是唯一取胜之道，而你写哪些规则，每一局都是不同的答案。拍卖只是主轴旁一条可选支线，走不走都不影响你写自己的规则。
