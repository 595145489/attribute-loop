AI 美术生产流程 v2 — aiart 集成版

Claude 负责决策和协调，aiart MCP 负责生成图片

第一步（只做一次）— 建立风格圣经
① 你描述大方向
"暗黑风格，组件图标"
→
② Claude 构造测试 prompt
positive + negative prompt
→
③ aiart.create_image_task
并行生成 4 张，等待完成
→
④ 浏览器展示 4 张
你选一张 / 迭代
→
⑤ 固化 style-bible.md
prompt前缀 + negative + 参考图 ID
风格圣经里存什么： positivePrompt 前缀 · negativePrompt · artSpecIds（art pack ID）· 第一批好图的 imageId（用于后续 referenceImages）
第二步（每次生产会话）— 单资源生产循环
① 读取清单
选一个"待生成"资源，读取描述
→
② 构造 prompt
风格圣经前缀 + 本资源描述
→
③ aiart × 4 并行
create_image_task × 4
referenceImages 传风格锚定图
→
④ 浏览器选图
4 张并排展示，你点选
→
⑤ 保存 + 更新清单
存到 resources/
manifest 标 ✓
不满意时： 你给文字反馈（"太亮了"/"换成圆形"） → Claude 调整 prompt → 重新调 aiart × 4 → 再选。
一致性保障： 用 referenceImages 把之前确认过的好图传进去做风格参考，新图自然跟老图搭。
aiart MCP 调用明细
生成阶段
// 并行调 4 次
create_image_task({
  taskType: "general",
  positivePrompt:
    [风格圣经] + [资源描述],
  negativePrompt: [禁止项],
  referenceImages: [锚定图],
  aspectRatio: "1:1",
  waitForCompletion: true
})
可选增强
🎨 artSpecIds — 锁定 art pack，最强风格一致性
🖼️ referenceImages — 传入已确认的好图做风格参考
⬆️ creativeUpscale — 选完后放大提升细节
✂️ removeBackground — sprite 类资源自动去背景
目录结构
resources/
  ├── icons/      # 组件图标 (heal.png, shield.png…)
  ├── sprites/
  │   ├── player/
  │   └── enemies/
  ├── ui/         # 面板/按钮/背景
  └── tiles/      # 轨道 tile、祭坛

docs/art/
  ├── style-bible.md
  └── asset-manifest.md
