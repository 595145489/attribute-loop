AI 美术生产流程 — 完整设计

方案 A：风格圣经 + 资源清单 + 浏览器审核

第一步（只做一次）

建立风格圣经

流程
① 你描述大方向（暗黑/像素/扁平…）
↓
② Claude 生成 4 张风格测试图
↓
③ 你在浏览器选一张 / 给反馈迭代
↓
④ 固化为 style-bible.md（永久基准）
style-bible.md 内容示例
# 风格基准 prompt 前缀
dark fantasy roguelike icon,
flat vector style, 128x128,
muted jewel tones with glow,
transparent background,
no photorealistic details,
consistent line weight

# 禁止项
no gradients, no text,
no realistic textures

# 色板
primary: #2c1654 #4a2080
accent: #00bcd4 #f39c12

第三步（每次生产会话）

单次资源生产循环

① 选取资源
从清单选一个"待生成"资源
→
② 构造 prompt
风格圣经前缀 + 资源描述
→
③ 生成 4 张
并行调 aiart API
→
④ 浏览器选图
你点选最满意的一张
→
⑤ 自动保存
存到正确路径，清单标 ✓
不满意？ 你可以给文字反馈 → Claude 调整 prompt → 重新生成 4 张 → 再选。循环直到满意为止。
